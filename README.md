# AoC



## Prerequisites

You will need [Leiningen][] 2.0.0 or above installed.

[leiningen]: https://github.com/technomancy/leiningen

## run

`lein repl`

```
user=> (ns aoc.server)
aoc.server=> (start)
```
## target_pressure

```
curl http://localhost:50005/target_pressure
{
  "ToExchange": {
    "Continue_mesaurement.Bool": false
  }
}
```
